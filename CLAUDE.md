## What this project is

This is the **agent-facing portal** for Truth in Real Estate. It will live at **https://tire-agents.fly.dev**. This site is for real estate agents who receive referrals from Kati.

**This project is not yet built out.** It is planned for a future phase.

**This is NOT the CRM.** The CRM (internal tool) is a separate repo called `truth-in-real-estate`.
